# Module greenhouse-fan1

A Viam module for temperature-controlled greenhouse fan automation on Raspberry Pi. It exposes a `Switch` component that monitors a temperature sensor and automatically toggles a relay-connected fan to maintain the configured temperature range.

## Models

This module provides the following model(s):

- [`viam:greenhouse-fan1:greenhouse-fan1`](viam_greenhouse-fan1_greenhouse-fan1.md) - Temperature-controlled fan switch using GPIO relay on Raspberry Pi
